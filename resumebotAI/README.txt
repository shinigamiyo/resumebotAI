Проект: простой телеграм бот на aiogram. Команды: /start показывает подсказку, /resume принимает pdf или docx резюме, чистит текст через pdfminer.six и выводит по страницам с датой, /search присылает три демо вакансии с кнопками.

Как проверить: нужен python 3.9 или новее. Создай окружение python -m venv .venv и активируй. Установи зависимости командой pip install -r requirements.txt.

Настройка токена: создай файл .env и запиши BOT_TOKEN=твой_токен или измени существующий файл.

Запуск: python bot.py. В телеграме нажми Start, проверь /resume на pdf и docx, посмотри что текст приходит страницами, запусти /search чтобы увидеть заглушки.

Что сделал: настроил парсинг pdf через pdfminer.six с запасным режимом PyPDF2, нормализовал текст без разрывов, добавил разбиение ответа по лимиту телеграма и шапку с датой, вынес токен в .env, добавил демо вакансии и подсказку /start.
